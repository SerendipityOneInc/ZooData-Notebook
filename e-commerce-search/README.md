# E-Commerce Search

Notebook demos for APIClaw e-commerce search APIs.

| Notebook | Description | Open |
| --- | --- | --- |
| `product_search.ipynb` | Search fashion products with natural-language text queries, brand filters, price filters, and retailer filters. | [![Open In Colab](https://colab.research.google.com/assets/colab-badge.svg)](https://colab.research.google.com/github/SerendipityOneInc/APIClaw-Notebook/blob/main/e-commerce-search/product_search.ipynb) |
| `image_search.ipynb` | Search visually similar fashion products from an image URL — with detection, bounding boxes, text guidance, "Shop the Look" pipeline, and direct SigLIP2 embeddings/similarity. Powered by [ZooClaw-FashionSigLIP2](https://huggingface.co/srpone/zooclaw-fashionsiglip2) ([paper](https://arxiv.org/abs/2606.27708)) — best open-source fashion VLM, with an even more robust commercial checkpoint behind the API. | [![Open In Colab](https://colab.research.google.com/assets/colab-badge.svg)](https://colab.research.google.com/github/SerendipityOneInc/APIClaw-Notebook/blob/main/e-commerce-search/image_search.ipynb) |
